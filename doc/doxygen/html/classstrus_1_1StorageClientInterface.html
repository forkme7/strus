virtual void strus::StorageClientInterface::defineStoragePeerInterface ( const StoragePeerInterface *  storagePeer,
bool  doPopulateInitialState = true 
)
pure virtual

Define the interface to use for creating peer storage transaction objects to populate global statistic changes to other peer storages (in case of a distributed index)

Parameters
[in]storagePeerreference to storage peer interface (owned by caller)
[in]doPopulateInitialStateflag that is set to true in a normal startup and to false in the case of a system recovery after a crash. The flag set to false tells the storage to populate its stored terms with a df change of 0 to the other peers in the initialization phase. After recovery restart the node should reinitialize its global statistics without changing the state of the other peers.
