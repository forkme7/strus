About

The open source project strus provides a collection of C++ (C++98) libraries and command line tools for building a competitive full-text search engine. The strus search engine can be build using any key value store database that provides an upper bound seek function for the stored key/value pairs. Currently there exists an implementation based on the LevelDB library.
The name strus is an acronym for structured search derived from the capability to identify complex structures in text and to extract parts of matches for further processing. You might for example search for a sequence of two words with one of a set of verbs in the same sentence that forms a nominal phrase from which the subject noun is extracted as part of the answer.
Besides finding structures, strus is of course also able to do classical information retrieval with a statistical, resp. stochastical (bag of words model) or a proximity weighting scheme.
