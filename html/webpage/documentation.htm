Interface documentation

Strus provides two classes of interfaces with diametric objectives. One is the C++ interface that is the base of the implementation and the other is the interface for language bindings with an implementation that wraps to the C++ interface.

C++ interfaces

  1. strus core (storage and query evaluation)
  2. strus analyzer
  3. strus module loader
  4. strus RPC client

Language binding interface

The language binding interface provides an interface to the strus core and the analyzer. Depending on how the context object is initialized, it uses either the module loader or it acts as proxy that issues its method calls via RPC to a running RPC server.
