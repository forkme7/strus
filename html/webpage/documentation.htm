Interface documentation

Strus provides two classes of interfaces with diametric objectives. One is the C++ interface that is the base of the implementation and the other is the interface for language bindings with implementations that wrap to the C++ interface.

C++ interfaces

  1. strus core (storage and query evaluation)
  2. strus analyzer
  3. strus module loader
  4. strus RPC client

Language binding interfaces

The language binding interfaces are for scripting languages calling strus functions. Language bindings can call strus functions directly or they can call functions on objects residing on a server via RPC. The documentation of the language bindings can be found here.
